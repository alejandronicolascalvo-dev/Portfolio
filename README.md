## 🚀 Proyectos Destacados

### 1. SkyOps - Dashboard de Operaciones Aéreas (OCC)
Un centro de control digital diseñado para la gestión de flotas y tripulaciones en tiempo real, resolviendo la necesidad de visibilidad inmediata en una base operativa.
*   **Temática:** Monitorización aeronáutica, optimización de disponibilidad y control de estados (Ready, Flying, Maintenance, SBY).
*   **Tecnologías:** HTML5, CSS3 (Modern UI) y JavaScript Vanilla (ES6).
*   **Funcionalidades Clave:** Dashboard completamente reactivo con actualización automática de KPIs, filtrado dinámico mediante buscador inteligente y lógica de disponibilidad para tripulaciones.
*   **Repositorio:** [Ver código de SkyOps](https://github.com/alejandronicolascalvo-dev/Portfolio) 

### 2. Agente IA Conversacional y Gobernanza Cloud (TechSoluciones)
Diseño, despliegue y evaluación automatizada de un asistente virtual de atención al cliente utilizando arquitecturas avanzadas en la nube.
*   **Temática:** Inteligencia Artificial, Ingeniería de Prompts y mitatigación de alucinaciones en entornos corporativos.
*   **Tecnologías:** Azure AI Foundry (Azure AI Studio), gpt-4o-mini / GPT-4.1, Azure AI Search.
*   **Hitos Técnicos:** 
    *   Inyección de un *System Message* estricto con políticas anti-jailbreak y control de tono corporativo.
    *   Evaluación automática del rendimiento basada en métricas de *Task Completion*, *Fundamentación* y *Coherencia*.
    *   Análisis y resolución de restricciones de infraestructura real (cuotas TPM/RPM e incompatibilidades estructurales de esquemas de datos .CSV vs .JSONL).

### 3. Iberia Connect (Proyecto Intermodular)
Aplicación web de tipo *One-Page* orientada a la gestión integrada de recursos técnicos y rutas del sector de la aviación.
*   **Tecnologías:** PHP, MySQL, HTML5, CSS3 y JavaScript.
*   **Funcionalidades Clave:** Panel interactivo con información técnica de aeronaves, gestión de rutas con meteorología simulada y navegación inmersiva mediante efecto Parallax.
*   **Entorno de desarrollo:** Desplegado localmente mediante XAMPP.
