### 1. Iberia Connect (Proyecto Intermodular)
Este es mi proyecto principal actual, diseñado como una aplicación web de tipo **One-Page** orientada al sector de la aviación.

**Temática:** Gestión de flota, destinos y recursos técnicos.
**Tecnologías:** PHP, MySQL (Base de datos relacional), HTML5 y CSS3.
**Funcionalidades Clave:**
    Panel interactivo con información técnica de aeronaves (modelo, autonomía, capacidad, antigüedad y estado).
    Gestión de rutas y destinos (Ciudad, código IATA, distancia y meteorología simulada).
    **Efecto Parallax:** Implementación de capas de fondo (nubes y paisajes) para una navegación inmersiva.
    **Entorno de desarrollo:** Desplegado localmente con **XAMPP**.
